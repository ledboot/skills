## API Operations

### Files
- Upload file: `/v1/files/upload`

### Lyrics
- Generate lyrics: `/v1/lyrics/generate`
- Extend lyrics: `/v1/lyrics/extend`

### Song
- Generate song: `/v1/song/generate`
- Query task: `/v1/song/query/{task_id}`
- Vocal Cloning: `/v1/vocal/clone`
- Extend song: `/v1/song/extend`
- Recognize song: `/v1/song/recognize`
- Describe song: `/v1/song/describe`
- Stem song: `/v1/song/stem`

### Instrumental
- Generate instrumental: `/v1/instrumental/generate`
- Query task: `/v1/instrumental/query/{task_id}`

### Uploads
- Create upload: `/v1/uploads/create`
- Add upload part: `/v1/uploads/add`
- Complete upload: `/v1/uploads/complete`

### Text to Speech
- Create speech: `/v1/tts/generate`
- Create podcast: `/v1/tts/podcast`

### Account
- Query billing: `/v1/account/billing`
