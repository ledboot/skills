# AI Tools Skills

A collection of AI Tools Skills.

## Project Introduction

This repository contains various skills that enhance AI assistants with specialized capabilities like music generation, audio processing, lyrics creation, and more. Each skill is self-contained with comprehensive documentation and API references.

## Available Skills

| Skill | Description |
|-------|-------------|
| [mureka](skills/mureka/) | AI music generation and audio processing - song generation, lyrics, instrumentals, vocal cloning, TTS, podcast, stem separation |

## Installation

### Install via skills.sh

```bash
npx skills add https://github.com/ledboot/skills
```
